Soulful Skincare Guide has a simple yet unique logo to give it unique branding for users to recognize it. Good contrast is achieved by using black text on light backgrounds, and white colored text on the hero image for readability and to grab the user’s attention. Images of real people are used to invoke emotion and engagement. “Arial” font is used on all the site because it’s easily readable. The website has a light blue and white minimalistic, as the mind associates the color blue with feelings of calmness and relaxation. The website uses HTML to separate content logically (e.g., headers, paragraphs, sections). The site is designed to maximize developer maintainability for future upgrades by having comments placed in the HTML, CSS, and JavaScript code to explain what it does for the website. The consistent structure of the navigation bar, hero section, and footer are applied to all the pages, making it easy to add new to pages seamlessly. Styling is maintained in a single CSS file which makes it easier to add new pages and make styling changes to the whole website in one place. “Cascading” is used in CSS so that lower level elements ‘inherit property values from higher level elements, reducing code duplication and making it easier to make changes. Alt text is used for all images in case they can’t be displayed and for compatibility for users that use screen readers.

Home page

The home page has a “Register now” button, that when clicked scrolls
the page down to the registration form, the button is a deeper blue than
the rest of the blue sections used in the website to grab the user’s attention
and call them to perform the action. The registration form has four fields for
the user to fill, if the user leaves a section empty the form does the validation
to tell the user to fill it, and if the format of the user’s input is not a format that
the form accepts, it tells them exactly how to put the right information.
When the user registers successfully, the screen scrolls up to a greeting
message is displayed to them under the hero section. Methods are used in
the JavaScript code to help other developers to extend functionality and maintain
the website, meaningful variable and method names are used as well.
The home page gives an overview of the subtopics the website contains
and what the website is about. A quote is included from a user who says the
website has helped improve their skin and mental health, encouraging users to
visit again and try the advice given for a long time.

Skincare basics page

In the hero section, there is a blue button which links to a skincare quiz which
lets users find out what their skin type is so that they can benefit from the
advice given for their specific skin type. An overview of the page encourages
the user to keep reading as it raises awareness on the importance of knowing
your skin type. Consistent styling by using one class to style all the images,
texts, and lists reduces the amount of duplicated code and lists organize
the information into readable sections. Images of how different skin types
help the user get reassured that skin problems are a normal occurrence,
which makes them feel seen and encourages them to use the website again
to learn more. The best skin products and why they should be used for each
skin type are listed in an organized manner.

Mindfulness page

Images of real people in a relaxed state make users subconsciously feel
calmness and serenity while reading, this encourages them to revisit the page.
Consistent styling is used to keep information organized and numbered list points help
keep the user engaged while reading.

Lifestyle page

Users are introduced to the topic of lifestyle and its effects on skin, raising their
awareness and increasing their curiosity to see what the rest of the page offers.
An image of a woman eating healthy food makes users want to eat the same way,
the best vitamins for skin health are mentioned and the foods that contain them
are listed, along with how they benefit the skin and mental wellbeing.
